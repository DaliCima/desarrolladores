complemento de notarios públicos.

 

Complemento para el manejo de la enajenación de bienes inmuebles.

Puedes descargar la documentación y definición del complemento en el sitio del SAT.

Datos del complemento de notarios públicos

A continuación una descripción y ejemplo de cada atributo en el complemento. La declaración de nameSpace es a nivel primario en nodo Comprobante. xmlns:notariospublicos="http://www.sat.gob.mx/notariospublicos"

  • NotariosPublicos

    • version : 1.0
      Atributo requerido para la expresión de la versión del complmemento, valor fijado en "1.0".
    • DescInmuebles
    • DescInmueble
      • TipoInmueble : 04
        Atributo requerido para la expresión del tipo de inmueble enajenado conforme al catálogo publicado en el portal del SAT en internet.
      • Calle : Dr. José María Coss
        Este atributo requerido sirve para precisar la avenida, calle, camino o carretera donde se ubica el inmueble.
      • NoExterior : 900
        Este atributo opcional sirve para expresar el número particular en donde se da la ubicación del inmueble en una calle dada.
      • NoInterior : 3A
        Este atributo opcional sirve para expresar información adicional para especificar la ubicación cuando calle y número exterior (noExterior) no resulten suficientes para determinar la ubicación precisa del inmueble..
      • Colonia : Centro
        Este atributo opcional sirve para precisar la colonia en donde se da la ubicación del inmueble cuando se desea ser más específico en casos de ubicaciones urbanas.
      • Localidad : Monterrey
        Atributo opcional que sirve para precisar la ciudad o población donde se da la ubicación del inmueble.
      • Referencia : Entre Matamoros y Zaragoza
        Atributo opcional para expresar una referencia adicional de ubicación del inmueble.
      • Municipio : Monterrey
        Atributo requerido que sirve para precisar el municipio o delegación (en el caso del Distrito Federal) en donde se da la ubicación del inmueble.
      • Estado : 05
        Entidad Federativa donde se ubica el inmueble conforme al catálogo publicado en el portal del SAT en internet.
      • Pais : MXN
        Atributo requerido que sirve para precisar el país donde se da la ubicación, conforme al catálogo publicado en el portal del SAT en internet.
      • CodigoPostal : 64000
        Atributo requerido que sirve para asentar el código postal en donde se da la ubicación del inmueble.
    • DatosOperacion
    • NumInstrumentoNotarial : 456998
      Atributo requerido que indica el número del instrumento Notarial donde consta la operación.
    • FechaInsNotarial : 2014-06-10
      Atributo requerido que indica la fecha de firma del instrumento Notarial.
    • MontoOperacion : 250000.00
      Atributo requerido para expresar el monto de la contraprestación o valor de la operación.
    • Subtotal : 200000.00
      Atributo requerido para expresar el subtotal de la contraprestación o su valor en la operación.
    • IVA : 0.00
      Atributo requerido para expresar el IVA de la contraprestación o su valor en la operación.
    • DatosNotario
    • CURP : PLCE800419HCLJRD02
      Atributo requerido para expresar la CURP del notario.
    • NumNotaria : 321
      Atributo requerido para indicar el número de la Notaria que realizar la operación.
    • EntidadFederativa : 05
      Entidad Federativa donde se ubica la Notaria conforme al catálogo publicado en el portal del SAT en internet.
    • Adscripcion : (Adscripción del Notario)
      Atributo opcional que expresa el señalamiento del notario a la plaza a la que se encuentra adscrito.
    • DatosEnajenante
    • CoproSocConyugalE : No
      Atributo requerido que expresa si es copropiedad o sociedad conyugal.
    • DatosUnEnajenante
      • Nombre : José
        Atributo requerido para expresar el nombre del enajenante.
      • ApellidoPaterno : Hernández
        Atributo requerido para expresar el apellido paterno del enajenante.
      • ApellidoMaterno : Hernández
        Atributo requerido para expresar el apellido materno del enajenante.
      • RFC : AAAA010101AAA
        Atributo requerido para la Clave del Registro Federal de Contribuyentes sin guiones o espacios, del enajenante.
      • CURP : AAAA010101HCLJND07
        Atributo requerido para expresar la CURP del enajenante.
    • DatosEnajenantesCopSC
      • DatosEnajenanteCopSC
      • Nombre : ----
        Atributo requerido para expresar el nombre (s) de cada enajenante.
      • ApellidoPaterno : ----
        Atributo opcional para expresar el apellido paterno de cada enajenante.
      • ApellidoMaterno : ----
        Atributo opcional para expresar el apellido materno de cada enajenante.
      • RFC : ----
        Atributo requerido para la Clave del Registro Federal de Contribuyentes sin guiones o espacios, correspondiente a cada enajenante.
      • CURP : ----
        Atributo opcional para expresar la CURP de cada enajenante.
      • Porcentaje : ----
        Porcentaje que le corresponde en la copropiedad a cada enajenante.
    • DatosAdquiriente
    • CoproSocConyugalE : Si
      Atributo requerido que expresa si es copropiedad o sociedad conyugal
    • DatosUnAdquiriente
      • Nombre : ----
        Atributo requerido para expresar el nombre, denominación o razón social del adquiriente.
      • ApellidoPaterno : ----
        Atributo opcional para expresar el apellido paterno del adquiriente.
      • ApellidoMaterno : ----
        Atributo opcional para expresar el apellido materno del adquiriente.
      • RFC : ----
        Atributo requerido para la Clave del Registro Federal de Contribuyentes sin guiones o espacios, del adquiriente.
      • CURP : ----
        Atributo opcional para expresar la CURP del adquiriente.
    • DatosAdquirientesCopSC
      • DatosAdquirienteCopSC
      • Nombre : Constructora ABC SA de CV
        Atributo requerido para expresar el nombre, denominación o razón social de cada adquiriente.
      • ApellidoPaterno : ----
        Atributo opcional para expresar el apellido paterno de cada adquiriente.
      • ApellidoMaterno : ----
        Atributo opcional para expresar el apellido materno de cada adquirientes.
      • RFC : AAA010101AAA
        Atributo requerido para la Clave del Registro Federal de Contribuyentes sin guiones o espacios, correspondiente a cada adquiriente.
      • CURP : ----
        Atributo opcional para expresar la CURP de cada adquiriente.
      • Porcentaje : 50.00
        Porcentaje que le corresponde en la copropiedad a cada adquiriente.

* Los campos en negritas indican atributos obligatorios

Ejemplo

<cfdi:Complemento>
  <notariospublicos:NotariosPublicos Version="1.0">
      <notariospublicos:DescInmuebles>
          <notariospublicos:DescInmueble TipoInmueble="04" Calle="Dr. José María Coss"
              NoExterior="900" NoInterior="3A" Colonia="Centro" Municipio="Monterrey"
              Estado="05" Pais="MEX" CodigoPostal="64000"/>
      </notariospublicos:DescInmuebles>
      <notariospublicos:DatosOperacion NumInstrumentoNotarial="456998"
          FechaInstNotarial="2014-06-10" MontoOperacion="250000.00" Subtotal="200000.00"
          IVA="0.00"/>
      <notariospublicos:DatosNotario CURP="TICE700419HCLJND07" NumNotaria="321"
          EntidadFederativa="05" Adscripcion="(Adscripción del Notario)"/>
      <notariospublicos:DatosEnajenante CoproSocConyugalE="No">
          <notariospublicos:DatosUnEnajenante Nombre="José" ApellidoPaterno="Hernández"
              ApellidoMaterno="Hernández" RFC="AAAA010101AAA" CURP="AAAA010101HCLJND07"/>
      </notariospublicos:DatosEnajenante>
      <notariospublicos:DatosAdquiriente CoproSocConyugalE="Si">
          <notariospublicos:DatosAdquirientesCopSC>
              <notariospublicos:DatosAdquirienteCopSC Nombre="Constructora ABC SA de CV"
                  RFC="AAA010101AAA" Porcentaje="50.00"/>
              <notariospublicos:DatosAdquirienteCopSC Nombre="Constructora XYZ SA de CV"
                  RFC="AAA010101AAA" Porcentaje="50.00"/>
          </notariospublicos:DatosAdquirientesCopSC>
      </notariospublicos:DatosAdquiriente>
  </notariospublicos:NotariosPublicos>
</cfdi:Complemento>
